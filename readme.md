# Väderapp
## Beskrivning
En väderapp som jag byggt med HTML, SASS och TypeScript. Jag använde mig av axios för att hämta API. 

Användaren kan söka efter valfri land eller stad, och temperaturen kommer skrivas ut i DOM:en.
